3.What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the DOM that involves attaching a single event listener to a parent element instead of individual listeners to child elements. When an event occurs on a child element, it bubbles up to the parent, allowing the parent to handle the event.
